Moment tensor entries represented in spherical coordinates. Six distinct moment tensor entries are used to form 3 x 3 symmetric tensor in spherical coordinates. These values are kept in sync with Strike / Dip / Rake Moment tensor entries represented in cartesian coordinates. Six distinct moment tensor entries are used to form 3 x 3 symmetric tensor. These values are kept in sync with Strike / Dip / Rake Strike, dip, and rake are used to describe earthquake Focal Mechanisms. These values are used to compute symmetric moment tensor and show Double Couple view. Double Couple view created by using JavaScript version of patch_meca.m file in Mirone source code ( http://w3.ualg.pt/~jluis/mirone/) - a MATLAB-based framework The code used for full 3D moment tensor is based on Carl Tape's work. See http://www.giseis.alaska.edu/input/carl/ for papers and reference to Mathematica scripts.
