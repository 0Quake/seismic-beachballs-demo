Moment tensor entries represented in spherical coordinates (r-t-p (up-south-east) basis). Six distinct moment tensor entries are used to form 3 x 3 symmetric tensor in spherical coordinates. These values are used to calculate Strike / Dip / Rake for closest double couple. Strike, dip, and rake are used to describe earthquake Focal Mechanisms. These values are computed based on moment tensor entries in r-t-p (up-south-east) basis 3Dモーメントテンソルの表示機能は、Carl Tape による成果に基づいています。論文および Mathematica スクリプトのリファレンスについては、 http://www.giseis.alaska.edu/input/carl/を参照してください。
